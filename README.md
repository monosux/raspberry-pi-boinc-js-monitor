# Raspberry Pi BOINC JS Monitor

## About

Get current BOINC tasks, CPU load, and temperature in your command line.

![Example](assets/image.png)

## Installation

```
// using npm
npm i pi-boinc-monitor -g
// or yarn
yarn global add pi-boinc-monitor
```

## Run

```
pi-boinc-monitor
```
